ポートフォリオ
(Web)
y.kom　（本サイト）

概要

自身の経歴や制作物をまとめたポートフォリオサイトを作成しました。

ターゲット層

Web制作に関するお仕事に携わっている方々

目的・ゴール

現状のスキルや経歴を知ってもらう

デザイン

明るく柔らかな印象の青やグレーでシンプルにまとめた中に、曲線や繊細な線で描いたイラストを配置して柔らかな印象になるようにしました。暖色系で表現しない点にも自分らしさを込めています。レスポンシブ対応でデザインするにあたって、長すぎるコンテンツが多いと関心を削いでしまうと考え、スマートフォン表示ではLightboxを使用するよう工夫しました。

制作範囲・期間

企画：約3日、
ワイヤーフレーム：約1週間、
デザイン：約2週間、
コーディング：約1ヶ月

使用ツール

Illustrator、Photoshop、
Visual Studio Code
