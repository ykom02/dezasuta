ホームページを介して人と人をつなぐ

「うまく言葉に出来ないけど･･･」に続く大事な部分を一緒に
探して、訪れた人に伝わるホームページをつくります。

できること

ヒアリング

ホームページを作るための土台作りを大切にします。 現状の課題や目的の共有はもちろん、魅力や雰囲気まで届けられるホームページを作るには事前準備が欠かせません。 前職である管理栄養士の仕事で培った傾聴力を活かして取り組みます。

デザイン

スマートフォンデバイスを想定したレスポンシブ対応のWebデザイン制作を行うことが可能です。 お客様の好まれる雰囲気や同じ業界のデザインを調べながら知識をストックし、「訪れた人にわかりやすい」デザインを心がけています。

ツール：Illustrator、Photoshop

コーディング

HTML/CSSを使用しデザインを正しく再現します。またjQueryを用いて動きや効果のある魅力的なホームページを作成することも可能です。 見やすく管理しやすいコーディングを目指し、勉強中です。

ツール： Visual Studio Code
